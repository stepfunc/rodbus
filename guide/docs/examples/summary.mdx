The source repository contains example programs you can compile and run interactively. Each program demonstrates various API features using interactive console input.

As you read through the examples, you may notice blocks of code surrounded by `ANCHOR` tags.

```c
// ANCHOR: logging

code that configures the logging ...

// ANCHOR_END: logging
```

We extract these code blocks from example programs during guide generation, ensuring all documentation snippets are syntactically valid and current.
